# Penguin Data Analysis (Assignment 1)

This repository contains an Exploratory Data Analysis (EDA) of the Palmer Penguins dataset, completed for **Assignment 1: Dev environment, data loading, processing and viz**.

## Project Overview
This project uses Python to load, clean, analyze, and visualize physical measurements of three penguin species (Adelie, Gentoo, and Chinstrap). The analysis calculates summary statistics, handles missing values, and utilizes `seaborn` and `matplotlib` to generate data visualizations.

## Task 8: Reflection
The trickiest part of this assignment was writing the code for the grouped charts, particularly dealing with library warnings. However, I was genuinely surprised by how clearly the physical measurements separated the penguin species into distinct clusters on the scatter plot. A logical next step would be using this clean dataset to build a machine learning classification model that can automatically predict a penguin's species. For this project, I used an AI assistant (Gemini) to brainstorm ideas for the visualizations, and I am really happy that I got to practice and write the actual code myself.

## How to Run This Code
To reproduce this analysis on your local machine:

1. Clone this repository to your computer.
2. Create and activate a Python virtual environment.
3. Install the required libraries by running:
   `pip install -r requirements.txt`
4. Open the Jupyter Notebook file and run the cells sequentially. The `penguins.csv` dataset is included in the repository.
